?? 작업 스케줄러로 quizapp.exe 자동 실행 설정 (hongj 전용)
? 준비 사항
관리자 계정으로 로그인

quizapp.exe는 C:\Program Files\QuizApp\ 같은 관리자 폴더에 저장되어 있어야 안전함

?? 단계별 설정
1. 작업 스케줄러 열기
Win + S → "작업 스케줄러" 검색 → 실행

2. 새 작업 만들기
오른쪽 메뉴에서 "작업 만들기(Create Task)" 클릭

3. 일반 탭 설정
이름: QuizApp 자동 실행

설명: hongj 로그인 시 quizapp.exe 실행

사용자 또는 그룹: hongj 입력

보안 옵션:

?? "가장 높은 권한으로 실행" 체크

?? "사용자가 로그온할 때만 실행" 선택

4. 트리거 탭 설정
"새로 만들기(New)" 클릭

시작 조건: 로그온(Log on)

사용자: hongj

?? "작업을 활성화" 체크

5. 동작 탭 설정
"새로 만들기(New)" 클릭

동작: 프로그램 시작(Start a program)

프로그램/스크립트: C:\Program Files\QuizApp\quizapp.exe 경로 입력

6. 조건 및 설정 탭 (선택 사항)
조건: 모든 체크 해제 (절전 모드 관련)

설정:

?? "요청 시 작업을 다시 시작"

?? "작업이 실패하면 다시 시도"

7. 저장
"확인" 클릭

관리자 권한 요청 시 비밀번호 입력

?? hongj가 작업을 삭제하지 못하게 하려면?
작업 스케줄러는 관리자 권한이 있어야 수정/삭제 가능합니다

hongj는 표준 사용자이므로 작업 자체를 삭제하거나 수정할 수 없습니다

? 테스트 방법
hongj 계정으로 로그아웃 후 다시 로그인

quizapp.exe가 자동으로 실행되는지 확인

?? 보너스 팁
Alt+F4 차단, 작업 관리자 접근 제한까지 하면 완전한 키오스크 환경 구축 가능

작업 스케줄러는 복구도 쉬우며, Shell 교체보다 훨씬 안전하고 유연해요
